# Credit Card Early Risk Detection System  
A lightweight ML + rule-based system to detect early delinquency risk from credit card customers.

## What this project does

- Upload CSV/XLSX customer data  
- Predict delinquency risk (Low / Medium / High)  
- Generate:
  - Risk probability  
  - Behavioural risk flags  
  - Recommended actions  
  - Portfolio summary (KPIs)  
  - Colored Excel report  
  - Customer-level PDF report  

## Tech Stack

- Python  
- Streamlit  
- scikit-learn  
- Plotly  
- openpyxl  
- reportlab
